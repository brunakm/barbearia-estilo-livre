# Barbearia Estilo Livre 💈

Projeto desenvolvido como atividade acadêmica com o objetivo de criar um site institucional responsivo para uma barbearia, apresentando serviços, informações da empresa e formas de contato.

## 📋 Sobre o Projeto

A Barbearia Estilo Livre é uma página web desenvolvida para simular a presença online de uma barbearia moderna. O site foi criado com foco em organização visual, experiência do usuário e adaptação para diferentes dispositivos.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap

## ✨ Funcionalidades

* Página institucional
* Apresentação da barbearia
* Exibição dos serviços oferecidos
* Layout responsivo
* Navegação simples e intuitiva
* Adaptação para dispositivos móveis

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/brunakm/barbearia-estilo-livre.git
```

2. Abra o arquivo `index.html` em qualquer navegador.

## 📚 Conceitos Praticados

* Estruturação com HTML5

* Responsividade
* Organização de conteúdo institucional
